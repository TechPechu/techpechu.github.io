Nginx File Structure

Nginx conf deep dive

Steps to Host Custom Host

Step 1: Create a .conf folder under conf.d

cd /etc/nginx/conf.d
vim techpechu.com.conf

Step 2: Add below setting in the conf file

server{
server_name techpechu.com www.techpechu.com;
listen 80;
root /var/www/techpechu.com/html/;
}

Step 3: Providing access for directory to access from Internet

Restorecon -R -v /var/www

Step 4: Reload the Nginx

systemctl reload nginx

Testing :

You can test locally with a curl call to your locahost

curl --header "Host: techpechu.com" localhost
